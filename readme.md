# dir2opml
对某一文件目录，快速生成其目录结构`tree.txt`，同时提供大纲文件 `tree.opml`，方便导入脑图工具，比如`mindNode`。

## 快速开始

1. **安装**
    1. 安装命令行工具`tree`
        ```shell
        brew install tree
        ```
    2. 安装 `dir2opml`
        ```shell
        npm i -g dir2opml
        ```
